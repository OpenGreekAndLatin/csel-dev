#Welcome to **csel-dev**!

##About
This repository hosts digital versions of the public-domain volumes of the _[Corpus Scriptorum Ecclesiasticorum Latinorum](http://www.csel.eu/)_ (CSEL).

These XML files were produced by data entry company [Jouve](http://www.jouve.com/en), who OCR'd, corrected and encoded each volume of CSEL in accordance with the latest [EpiDoc](http://sourceforge.net/p/epidoc/wiki/Home/) standards.
While adequate, more can and should be done to improve the XML (the -dev(elopment) suffix indicates the repository is open to improvement).

A Wikipedia link to this repository can be viewed [here](https://en.wikipedia.org/wiki/Corpus_Scriptorum_Ecclesiasticorum_Latinorum#External_links).

The following is a breakdown of the volumes (list adapted from the [CSEL official webpage](http://www.csel.eu/?id=19)):
