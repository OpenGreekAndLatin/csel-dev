        <h1>
<a id="welcome-to-csel-dev" class="anchor" href="#welcome-to-csel-dev" aria-hidden="true"><span class="octicon octicon-link"></span></a>Welcome to <strong>csel-dev</strong>!</h1>

<h2>
<a id="about" class="anchor" href="#about" aria-hidden="true"><span class="octicon octicon-link"></span></a>About</h2>

<p>This repository hosts digital versions of the public-domain volumes of the <em><a href="http://www.csel.eu/">Corpus Scriptorum Ecclesiasticorum Latinorum</a></em> (CSEL).</p>

<p>These XML files were produced by data entry company <a href="http://www.jouve.com/en">Jouve</a>, who OCR'd, corrected and encoded each volume of CSEL in accordance with the latest <a href="http://sourceforge.net/p/epidoc/wiki/Home/">EpiDoc</a> standards.
While adequate, more can and should be done to improve the XML (the -dev(elopment) suffix indicates the repository is open to improvement).</p>

<p>A Wikipedia link to this repository can be viewed <a href="https://en.wikipedia.org/wiki/Corpus_Scriptorum_Ecclesiasticorum_Latinorum#External_links">here</a>.</p>

<p><strong>How to download individual files from this page</strong><br>
Clicking on the file's link will take you to the file in the repository. If you wish to download the TEI-XML of the file, right click on the link and choose "Save link as..."</p>
